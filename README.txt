GPT Telegram Бот с кнопками, Google Таблицей и GPT-режимами

1. Установка:
    pip install -r requirements.txt

2. Настрой файл .env (можно скопировать из .env.example)

3. Запуск:
    python main.py

4. Функции:
    - Кнопки: "Услуги", "Оставить номер", "Контакты"
    - Сбор номера телефона (сохраняется в Google Таблицу)
    - GPT-режим: mock (шаблонные ответы) или real (настоящий GPT)
